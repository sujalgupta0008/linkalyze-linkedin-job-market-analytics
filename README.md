# 📌 Project Overview

The job market is evolving rapidly, making it difficult for students, professionals, and recruiters to identify hiring trends, salary expectations, and the most valuable skills.

**LinkaLyze** is an end-to-end data analytics project that analyzes more than **124,000 LinkedIn job postings (2023–2024)** to uncover meaningful insights about hiring patterns, salaries, remote work, industries, companies, and skill demand.

The project follows the complete data analytics workflow—from data cleaning and exploratory data analysis (EDA) to interactive dashboard development using Power BI.

---

# 🎯 Objectives

- Perform comprehensive data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyze salary distributions across different roles
- Identify the most in-demand technical skills
- Compare Remote, Hybrid, and On-site job opportunities
- Discover top hiring companies and industries
- Build an interactive Power BI dashboard
- Generate actionable insights for students, job seekers, and recruiters

---

# 📂 Dataset Information

- **Source:** LinkedIn Job Postings Dataset
- **Time Period:** 2023–2024
- **Total Records:** 124,000+
- **Columns:** 31+
- **Includes:**
  - Company Information
  - Job Titles
  - Salary Details
  - Skills
  - Benefits
  - Employment Type
  - Experience Level
  - Location
  - Industry
  - Remote/Hybrid Status
